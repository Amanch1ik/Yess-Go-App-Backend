ИНСТРУКЦИЯ ПО ЗАПУСКУ FRONTEND (ADMIN PANEL)
============================================

1. Откройте PowerShell в директории: Yess-Money---app-master\admin-panel

2. Установите зависимости (если еще не установлены):
   npm install

3. Создайте .env файл (если его нет):
   echo "VITE_API_URL=http://localhost:8000" > .env

4. Запустите dev сервер:
   npm run dev

Admin Panel будет доступен на: http://localhost:3001

ВАЖНО: Убедитесь, что Backend запущен на http://localhost:8000 перед запуском Frontend!
