# Yess!Go - Партнерская панель

Партнерская панель для системы лояльности Yess!Go.

## Запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

Приложение будет доступно на `http://localhost:3002`

## Структура

- `/` - Главная (Dashboard)
- `/profile` - Профиль партнера
- `/locations` - Локации партнёра
- `/promotions` - Акции и сторисы
- `/transactions` - Транзакции
- `/employees` - Сотрудники
- `/integrations` - Интеграции
- `/billing` - Биллинг

## Особенности

- Анимированная кнопка удаления на всех страницах
- Современный дизайн в стиле Yess!Go
- Адаптивная верстка
- Интеграция с API
